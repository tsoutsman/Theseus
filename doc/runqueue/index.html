This crate contains the API of the RunQueue structure, Runqueue Structure should contain list of tasks with additional scheduling information depending on the scheduler. All crates except the scheduler should refer to this crate to access functions on RunQueue.

Functions

add_task_to_specific_runqueue
Adds the given Task reference to given core’s runqueue.

Returns the “least busy” core
