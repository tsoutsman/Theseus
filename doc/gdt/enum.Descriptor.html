pub enum Descriptor {
    UserSegment(u64),
    SystemSegment(u64, u64),
}
Expand description

The two kinds of descriptor entries in the GDT.

Variants§

§

UserSegment(u64)

UserSegment is used for both code and data segments, in both the kernel and in user space.

§

SystemSegment(u64, u64)

SystemSegment is used only for TSS.
